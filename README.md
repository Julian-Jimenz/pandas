# pandas
Este ejercicio práctico se realiza con el fin de aplicar los conocimientos adquiridos sobre la librería de Python (Pandas).
El archivo contiene 16 preguntas de negocio, las cuales se responden con base en un archivo CSV que incluye el informe de ventas y facturación de productos.
Este ejercicio pone a prueba la creatividad y la lógica para resolver las preguntas de manera efectiva.
